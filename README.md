# Unity-FPS
Prototype scripts with movement and combat for an fps parkour game

**Player movement**<br>
- Sliding
- Dashing
- Swinging
- Grappling
- Climbing
- Ledge grabbing
- Wallrunning

**Combat and other features**
- Enemy AI
- Guns
- Portals
- Draggable objects, pressure pads and doors
- Jump and bounce pads
